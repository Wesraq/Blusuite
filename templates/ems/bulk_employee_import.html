Instructions

How to Import Employees:
  • 1 Download Template: Click the button below to download the CSV template
  • 2 Fill Data: Open the CSV in Excel/Sheets and fill in employee details
  • 3 Save as CSV: Save the file in CSV format (not Excel .xlsx)
  • 4 Upload: Upload the completed CSV file below
  • 5 Review: Check the import results and fix any errors

CSV Format Reference

Column Name Required Description Example
email Required Employee email address (must be unique) john.doe@example.com
first_name Required Employee first name John
last_name Required Employee last name Doe
employee_id Optional Company employee ID EMP001
job_title Optional Job title/position Software Engineer
department Optional Department name IT
phone_number Optional Contact number +1234567890
date_hired Optional Hire date (YYYY-MM-DD format) 2024-01-15
⚠️ Important Notes:
  • All imported employees will receive a temporary password: TempPass123!
  • Employees should reset their password on first login
  • Duplicate email addresses will be skipped
  • The CSV file must use UTF-8 encoding
